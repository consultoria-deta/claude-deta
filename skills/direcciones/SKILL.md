# Direcciones — Libreta de Contactos DETA

## Para qué sirve

Resolver aliases y nombres informales a correos reales. **Google no conoce aliases — solo correos.** Antes de cualquier llamada a Calendar o Gmail, resolver aquí primero.

## Regla de oro

1. El usuario dice un nombre → buscar en esta libreta → usar el correo real
2. Si no está aquí → preguntar a Joel, **nunca inventar un correo**
3. Si Joel da un correo nuevo → agregarlo a esta libreta en el momento, antes de continuar

## Protocolo de actualización incremental

Los aliases **no se llenan de golpe** — se van agregando conforme Joel los menciona en conversación.

Cuando Joel use un nombre o alias nuevo:
1. Si el correo ya está en la libreta sin alias → agregar el alias a esa fila
2. Si el contacto no existe → preguntar el correo, luego agregar la entrada completa
3. Editar este SKILL.md inmediatamente con el alias nuevo — no esperar al cierre de sesión
4. Confirmar en una línea: *"Agregué [Alias] → correo@ejemplo.com"*

Ejemplos de triggers para actualizar:
- Joel dice "mándale a Maru de Sercom" → no está → preguntar correo → agregar
- Joel dice "Chatato también le dicen Humberto el chico" → agregar alias a hperezch@sapisa.com.mx
