## Cómo usar

1. El usuario menciona un estilo o referencia → leer el archivo correspondiente
2. Aplicar los tokens (colores, tipografía, spacing) del DESIGN.md elegido
3. Si es para DETA Web → `deta-brand` tiene prioridad; este skill es solo para trabajos externos o experimentación

### Cuándo combinar con deta-brand
- **No combinar** — son sistemas distintos e incompatibles en producción
- Para DETA Web: siempre `deta-brand`
- Para cliente X: este skill + el brief del cliente
